The files here contain information for the execution of the DoAP_PrEWS algorithm.

This is a DoA prediction and early warning algorithm for the model development of EEG signal data. Specifically developed for EEG based DoA analysis.

This work supports the methods published by Thomas Schmierer at UniSQ in "Prediction_of_Anesthesia_Depth_and_Early_Detection_of_State_Transitions_Using_EEG-Derived_Features"
